Σε αυτό το directory υπάρχει ο κώδικας που χρειάζεται ώστε να προσθέσουμε στο redis
υποστήριξη για utmem operations. Αφορά το unikernel redis το οποίο προσφέρεται από
τα rumprun-packages.
Τα command που προστίθενται είναι, tmemPut, tmemGet, tmemInval.

Σημείωση: το redis από την έκδοση 4.0.0 και ύστερα υποστηρίζει redis-module, ένα
ασφαλές API για on the fly εισαγωγή νέων εντολών/λειτουργιών, όμως η unikernel
έκδοση στηρίζεται στο 3.6.0, και είμαστε αναγκασμένοι να προσθέσουμε τις εντολές
στον πηγαίο κώδικα του redis. Μικρό το κακό, απλά είναι πιο σύνθετη η εισαγωγή.

Για να γίνει αυτό πρέπει αρχικά να γίνει build ένα vanilla redis-server unikernel,
όμως προτείνουν οι επίσημες οδηγίες στα rumprun-packages. Αυτό συμβαίνει ώστε να
κατέβουν από το internet τα αρχεία και να δημιουργηθούν τα σωστά directories.
Στην συνέχεια πρέπει να εκτελεστεί το script setFiles.sh  το οποίο τοποθετεί τα
κατάλληλα αρχεία στον πηγαίο κώδικα του redis. Επίσης αντιγραφει την tmem από το
directory rumprun-tmem-func. Στην συνέχεια ξαναδημιουργούμε το redis unikernel
και είμαστε έτοιμοι.

Στο αρχέιο scripts.txt υπάρχει shell scripts που χρειάζονται για το χτίσιμο
και εκτέλεση του server, όπως π.χ. πως να σεταριστεί η network device κλπ.

Η επικοινία με τον server γίνεται με την εντολή nc (η γενικά με οποιοδήποτε τρόπο που μπορεί
να επικοινωνήσει με network requests). Δεν υπάρει υποστήριξη για redis-client ως unikernel.

Ως χρήστες πρέπει να θέσουμε τα σωστά path στο setFiles.sh αλλά και στο build-src-Makefile
στην γραμμή 14.

Τα αρχεία είναι
1.redis.c
Δηλώνει τα properties των 3 νέων λειτουργιών, όπως όνομα και function που το αναλαμβάνει.
Γραμμή 287 έως 289
2.redis.h
Declaration των 3 συναρτήσεων. Γραμμή 1559 έως 1561.
3.Makefile
Είναι αντίγραφο του rumprun-packages Makefile, με διαφορά πως δεν ασχολείται με το
redownload των αρχείων. Επίσης κατά το make clean δεν σβήνει τους πηγαίους κώδικες,
αλλά μόνο τα compile aritfacts.
4.build-src-Makefile
Εσωτερικό Makefile που δημιουργεί το redis-server. Ενημερωμένο ώστε να συμπεριλαμβάνει
το αρχείο tmem-redis.c και το tmem_ops.c. Γραμμές 14, 118, 199 εως 201
5.tmem-redis.c
Implementation των 3 συναρτήσεων του redis που εκτελούν τα tmem operation. Χρησιμοποιεί το
tmem_ops.c για την εξυπηρέτηση αυτών. Τα ζευγάρια key-value θεωρούνται πως είναι πάντοτε C string
οπότε η ανταλλαγη μνήμης γίνεται με βάση αυτό, δηλαδή οποιοδήποτε key ή value πρέπει να μπορεί να
γίνει typecast σε *char, αλλιώς θα υπάρξει πρόβλημα κατά την εκτύπωση των αποτελεσμάτων.
